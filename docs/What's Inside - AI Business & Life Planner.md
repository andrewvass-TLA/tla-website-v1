## Part 2 — Bottom Block (Click-to-Reveal)

### Interaction Behavior

- Four dropdowns stacked vertically in the left column.
- Only one dropdown can be open at a time. Opening one closes the others.
- The image in the right column updates based on which dropdown is open.
- On initial load, Dropdown 1 is open by default with its corresponding image displayed.
- On mobile, the image appears inline directly below the active dropdown header.

## Accessibility

- Dropdowns must be keyboard-navigable (Enter/Space to expand).
- Use proper ARIA attributes (aria-expanded, aria-controls).
- Each image requires the alt text specified above.

---

## Optional Enhancement

Consider numbering the dropdowns (1, 2, 3, 4) to reinforce the sequence: Assess → Diagnose → Plan → Execute.
